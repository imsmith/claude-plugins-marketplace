# Claude Code Plugin Marketplace

Personal Claude Code plugins for BEAM development, infrastructure, document production, and session management.

## Plugins

- **beam-style-guard** — Enforces BEAM/Elixir style guide principles via automated checks and on-demand audits
- **codebase-forensics** — Forensic codebase analysis for inherited, undocumented, or post-acquisition codebases
- **infra-preferences** — Infrastructure tool preferences and best practices for Podman, CoreDNS, Caddy, wireguard, and eBPF monitoring
- **session-wrap** — Session wrap-up: summarize work, update memory, log session, and bookmark for resumption
- **tufte-press** — Convert markdown files to Tufte-style PDFs via pandoc + pdflatex

## Usage

```
claude plugins marketplace add imsmith/claude-plugins-marketplace
claude plugins install <plugin-name>@imsmith-marketplace
```
